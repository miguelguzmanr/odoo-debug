=======
Debugpy
=======

This module adds Debugpy integration in Odoo.

.. contents::
   :local:

Installation
============

To install this module, you must enable developer mode.

Usage
=====

#. Go to *Settings* or *Settings > General Settings*
#. Activate the developer mode
